# 🎥 Zoom Clone

A real-time video conferencing application built using React, Node.js, Express, Socket.io, and WebRTC. Users can create or join meeting rooms and communicate through live video streams.

## 🚀 Live Demo

🔗 Live App: https://zoom-clone-ktvr.onrender.com

## 📂 GitHub Repository

🔗 Repository: https://github.com/Mahendra-2173/Zoom-Clone

---

## ✨ Features

- 📹 Real-time video calling
- 👥 Multi-user meeting rooms
- 🔗 Shareable meeting links
- 🎤 Audio and video communication
- ⚡ Real-time signaling with Socket.io
- 🌐 WebRTC peer-to-peer connection
- 📱 Cross-browser support
- 🚀 Deployed on Render

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Mahendra-2173/Zoom-Clone.git
cd Zoom-Clone
```

### Install Dependencies

```bash
npm install --legacy-peer-deps
```

### Run Locally

```bash
npm run dev
```

Application will run on:

```text
http://localhost:8000
```

## 🔍 How It Works

1. User creates or joins a room.
2. Socket.io establishes signaling between participants.
3. WebRTC creates peer-to-peer connections.
4. Media streams are exchanged between connected users.
5. Participants can communicate in real time through video and audio.
